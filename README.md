# ICB by Adiyar - Централизованный образовательный портал

Полнофункциональный веб-портал для управления образовательным процессом с поддержкой мобильных устройств.

## 📱 Адаптивность

Сайт полностью адаптирован под все типы устройств:
- **Мобильные телефоны** (375px - 767px): Оптимизированный интерфейс с бургер-меню
- **Планшеты** (768px - 991px): Компактный дизайн с адаптивным макетом
- **Десктоп** (992px+): Полный функциональный интерфейс

## 🎨 Особенности

### Главная страница (`index.html`)
- Современный дизайн с темной и светлой темой
- Аутентификация (тестовые учетные данные)
- Быстрый доступ к сервисам
- Виртуальный чат-помощник
- Адаптивная навигация

### Журнал студента (`journal.html`)
- Расписание занятий
- Список задач с приоритетами
- Систематизированные заметки
- Обзор успеваемости
- Система уведомлений
- Мобильное боковое меню

### Журнал преподавателя (`teacher-journal.html`)
- Управление курсами
- Планирование расписания
- Обзор студентов
- Статистика класса

### Журнал студентов преподавателя (`teacher-students.html`)
- Таблица всех студентов
- Управление оценками
- Отслеживание посещаемости
- Назначение домашних заданий
- Фильтрация по группам и курсам

## 📱 Мобильная оптимизация

### Особенности мобильной версии
1. **Бургер-меню** - компактное боковое меню на экранах ≤600px
2. **Горизонтальная прокрутка таблиц** - для удобства просмотра больших данных
3. **Оптимизированные кнопки** - минимальная высота 44px для мобильных
4. **Адаптивные формы** - вертикальное расположение на мобильных
5. **Полноэкранные модалы** - на очень малых экранах

## 🚀 Быстрый старт

1. Откройте `index.html` в браузере
2. Используйте тестовые учетные данные для входа
3. Выберите роль (студент или преподаватель)
4. Изучите функциональность портала

## 🌙 Смена темы

Используйте кнопку переключения темы в верхнем левом углу:
- **Светлая тема** - белый фон, черный текст
- **Темная тема** - темный градиент, белый текст

Данные о выборе темы сохраняются в LocalStorage.

## 📊 Сохранение данных

Все данные сохраняются в LocalStorage браузера:
- Информация об аутентификации
- Оценки и отметки
- Заметки пользователя
- История чата
- Выбранная тема оформления
